Device EEPROM

Device EEPROM system ID: 0x21

Device EEPROM system provides read and write functionality to the EEPROM located in the actuator. By default EEPROM data are write protected. To write data, the user has to explicitly unlock the EEPROM by writing the key value to the lock register.

Note
Before accessing Device EEPROM on particular channel it is necessary to set active channel in Status system (register address 0x1008).
