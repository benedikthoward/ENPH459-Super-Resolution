Analog Input

Analog Input system base ID: 0x58

Once this system is activated, the input to the Signal Flow is determined by the analog input signals available to the driver. The analog inputs are sampled at controller frequency which is 20kHz. Depending on the active control mode configured by the Signal Flow Manager the maximum analog input range is mapped linearly to the maximum range of the corresponding unit set by the control mode by default. This can be changed by setting Look Up Table, which consists of input voltages and specific value of current or dpt. After setting LUT, specific register with type(current/dpt) must be set. Maximum size of table is 10 pairs.

Note
Full register address varies according to the channel.

For example:

ADC raw value register for channel two has address 0x5900

Active input type register for channel one has address 0x5801

The Analog input system supports the following vectors (vectors have their own address space):

Address Name Default Description Format and value Access
0x5[8-F]00 Voltages -> LUT table values 1.voltage max 10 * float read write
0x5[8-F]01 Current/dpt -> LUT table values 2.value(current/FP) max 10 & same # as voltages * float read write
0x5[8-F]02 Analog input cal.coefficients {0,1,0,0,0,0} 5th grade polynomial 6 * float (24 bytes) read write
Note
Setting Voltage vector must precede setting of Value vector.
