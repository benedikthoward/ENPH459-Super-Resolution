ICC4C  143-525-00
Linear Output

Linear output system base ID: 0xE8

Linear output system directly propagates current values to hardware and provides also some safety features.

Register Map:

Address Name Default Value Description Type Access
0xE[8-F]00 Current limit 0.5 or 2.0 * Limit in amperes where the output clamps (automatically set negative limit as well) float read write
0xE[8-F]01 Channel number N/A uint32 read only
0xE[8-F]02 Output current N/A Current current output in amperes float read only
0xE[8-F]03 Average limit period 0.5 Number of seconds to average current float read write
0xE[8-F]04 Average limit sum 0.5 Maximum allowed average current in amperes float read write
0xE[8-F]05 Cool-down period 10.0 Length of cool-down period in seconds float read write
0xE[8-F]06 Output disable false (0) Disable output signal bool read write
0xE[8-F]07 Calibr.coef.enable false (0) Enable polynomial approximation (calibration) bool read write
0xE[8-F]08 Current limit negative 0.5 or 2.0 * Negative current limit for asymetric operation float read write
0xE[8-F]09 Direct input value 0.0 * Current value directly propagated to DAC if output is disabled float read write

*depends on FW build and driver type

Note
Full register address varies according to the channel.

For example:

Current limit register of channel one has address 0xE800

Output current register of channel two has address 0xE902
