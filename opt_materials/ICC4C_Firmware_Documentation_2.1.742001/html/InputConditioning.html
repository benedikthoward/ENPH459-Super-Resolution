ICC4C  143-525-00
Input Conditioning

Input Conditioning system base ID: 0x98

Register Map:

Address Name Default Value Description Type Access
0x9[8-F]00 Gain 1.0 Multiplier of input float read write
0x9[8-F]01 Offset 0.0 Offset to input float read write
0x9[8-F]02 Transition rate 0.0 Limits change of output value per second (0.0 means no limit) float read write
0x9[8-F]03 Limit low 0.0 Low limit for input value (*) float read write
0x9[8-F]04 Limit high 0.0 High limit for input value (*) float read write
0x9[8-F]05 Limit enable false = 0 Enable input value limiting (*) boolean read write

(*) When setting register value condition limit_low < limit_high is being checked otherwise error code REGISTER_VALUE_OUT_OF_RANGE (Error Codes) is returned. The limiting feature will be enabled only if condition above is true.

Note
Full register address varies according to the channel.

For example:

Offset register of channel two has address 0x9901

Gain register of channel one has address 0x9800
