Vector Pattern Unit

Vector Pattern Unit system base ID: 0x68

The Vector Pattern Unit system is an input stage system that can used to propagate arbitrary vector values set by the user as input to the Signal Flow. The values must be saved in the Vector Pattern Memory. The Vector Pattern system has to be activated through the Signal Flow Manager. Unit type has to match to the configured control mode.
