ICC4C  143-525-00
Hall Feedback for Lyra

Hall Readout system ID: 0x2C

The hall readout system provides information about the actual position of actuator. It also includes the parameter registers for setup of the hall sensor. Caution - sensor readout has to be disabled while accessing registers!

Register Map:

0x2C06 Resolution 1 Bit resolution (0=16b, 1=17b, 2=18b, 3=19b) uint8 read write
