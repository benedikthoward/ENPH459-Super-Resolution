Vector Adapter

Vector Adapter system ID: 0x28

This system provides an adapter interface to manipulate system vectors using only set and get register commands. This is useful for SPI and I2C interfaces that support only register manipulation.

Register Map:

Address Name Default Value Description Type Access
0x2800 Vector data N/A Vector data to write or read. uint8[4] read write
0x2801 Vector address N/A Vector address ('System ID' + 'Vector ID') to write or read. uint16 read write
0x2802 Vector index N/A Index within vector to write or read uint16 read write
0x2803 Data size 4 Number of bytes to write or read. Can be between 1 and 4. uint8 read write
0x2804 Write N/A Writing any value to this register triggers writing.
After writing, vector index is automatically incremented by the data size. 		uint32 read write
0x2805 Read N/A Writing any value to this register triggers reading.
After reading, vector index is automatically incremented by the data size. 		uint32 read write
0x2806 Error flag 0 Contains the error flag returned by the previous write or read operation. uint32 read only
