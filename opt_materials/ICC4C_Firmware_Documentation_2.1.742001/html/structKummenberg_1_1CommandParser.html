Detailed Description

The message parser to command. This takes a bunch of bytes as a Buffer as input, copy them in internal FIFO and returns immediately. This design is required since the Serial code that'll connected to this needs to answer as fast as possible since it's called in interrupt handlers. Processing is done later on by the main loop.
