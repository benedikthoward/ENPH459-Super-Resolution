optoICC.registers.icc_registers.ICCDeviceTemperatureManager.GetDeviceTemperatureThreshold ( self)

Gets temperature threshold of the device connected to this channel.

If this threshold is reached the output of the driver is disabled to cool down. Temperature has to fall by 10 degrees so that the output is enabled again.
