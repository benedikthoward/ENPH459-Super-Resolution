ICC4C  143-525-00
Static Input

Static Input system base ID: 0x50

The static input stage system belongs to the input stage systems of the Signal Flow and is the one which is by factory default activated at start up. With this system the user can set constant values as input to the signal flow. The register which is propagated depends on the active control stage configured by the Signal Flow Manager.

Register Map:

Address Name Default Value Description Type Access
0x5[0-7]00 Current static input 0.0 Static input in amperes. Range depends on driver type float read write
0x5[0-7]01 OF static input 0.0 Static input in OF. Range differs for each mirror float read write
0x5[0-7]02 XY static input 0.0 Static input in XY coordinates. Range is from -1.0 to 1.0 float read write
0x5[0-7]03 Active static input type 0 Determined by active control stage system Unit Type read only
0x5[0-7]04 FP static input 0.0 Static input in DPT. For lenses only. float read write
0x5[0-7]05 Unitless value 0.0 Unitless value. Can be used for non specific value type. float read write
Note
Full register address varies according to the channel.

For example:

XY input register for channel two has address 0x5102

Current input register for channel one has address 0x5000
