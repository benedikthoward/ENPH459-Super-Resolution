ICC4C  143-525-00
Lens Compensations

Lens Compensations control system base ID: 0xC8

  • Current Mode - directly feeds through current
  • Focal Power Mode - applies 1st and 2nd order Temperature compensation corrections, as well as “Membrane Creep” corrections

At startup, the 9 drift parameters are read from the EEPROM.
At every time step when controlling the lens the following calculations have to be performed:

  • get desired focal power (sfp[n]), measured Temperature (T[n])
  • unit drift compensation, range: (0,1), captures dynamic behaviour described by T1up, T1down,T2,T3,T4
  • gain temperature compensation: G1, G2 and G3 need to be corrected for T[n]
  • scaled drift compensation: mapping of normalised dynamics to focal power (dpt) with G1’,G2’,G3’,G4, sfp[n]
  • difference sfp[n] – z[n] gives a focal power that will be used for the usual focal power mode (temperature compensation lookup)
  • from the lookup (5) we get a current value sc[n] that is set to the lens
