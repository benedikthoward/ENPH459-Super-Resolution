Detailed Description

template<size_t FixedSize>
class CircularBuffer< FixedSize >

A simple two head and size limited buffer AKA circular/ring buffer. This is used to store a contiguous stream of data, where the oldest item are discarded to store the new one.

Parameters
FixedSizeThe buffer size (should be a power of 2)
Warning
The number of stored element in this array is exactly (size - 1). This is a bit clever as it stores the len of the added element in the array first
